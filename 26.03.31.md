*콤보박스에서 화면에 보여줄 값이랑 실제로 쓰는 값이다를때  
DisplayMemberPath-화면에 보이는 값
SelectedValuePath- 선택했을 때 내부적으로 가져오는 값
SelectedValue- 실제변수에 들어가는 특정컬럼 값

동작흐름
ItemSource ->콤보 데이터 바인딩
DisplayMemberPath='컬럼명' -> 화면에 보여줄 컬럼
SelectedValuePath='컬럼명' ->실제 값
SelectedValue -> 내가 쓰는 값

*GetDataSet
DB 연결: SqlConnection 등을 열어서 DB에 접속합니다
명령 준비: 파라미터를 세팅합니다.
데이터 퍼오기: SqlDataAdapter를 사용해 DB의 ResultSet을 퍼옵니다.
바구니에 담기: 퍼온 데이터를 DataSet 객체의 Tables[0] 등에 채웁니다.
반환: 데이터가 꽉 찬 DataSet을 호출한 곳으로 돌려줍니다

GetDataSet(dbconname, "패키지.프로시저명", true, 컬렉션)
dbconname-연결 이름
패키지.프로시저명-디비 내부 주소
true-sql문장이 아니라 프로시저 이름이라고 선언, 만약 false면 select * from 같은 긴 쿼리문을 직적 써야한다
컬렉션- 변수 값들의 묶음
